# 🛒 Zepto E-Commerce Data Analysis using MySQL

## 📌 Project Overview

This project analyzes a Zepto e-commerce product dataset using **MySQL** to extract meaningful business insights. The project covers data exploration, data quality checks, KPI analysis, and business-focused SQL queries that help understand product performance, pricing, discounts, and inventory.

---

## 🎯 Project Objectives

- Explore and understand the dataset.
- Check for NULL values and ensure data quality.
- Analyze key business metrics (KPIs).
- Solve real-world business problems using SQL.
- Generate insights to support business decision-making.

## 📊 Business Questions Answered

### 1. Top 10 Products with the Highest Discount Percentage
Identified products offering the highest discounts to understand promotional strategies.

### 2. Out-of-Stock Products with the Highest MRP
Found expensive products that are currently unavailable in inventory.

### 3. Estimated Revenue by Category
Calculated the estimated revenue for each product category using selling price and available quantity.

### 4. Premium Products with Low Discounts
Identified products with an MRP greater than ₹500 and a discount of less than 10%.

### 5. Top 5 Categories with the Highest Average Discount
Determined which product categories offer the highest average discounts.

### 6. Best Value Products Based on Price per Gram
Calculated price per gram for products weighing at least 100 grams to identify the best-value products.

### 7. Product Weight Classification
Categorized products into **Low**, **Medium**, and **Bulk** weight groups using SQL CASE statements.

### 8. Total Inventory Weight by Category
Calculated the total available inventory weight for each product category.
